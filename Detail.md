# 🗒 게시판 홈페이지
---
##### 📆 작업 기간 : 6/5 ~ 6/8
##### 💻 사용 프로그램 : Eclipse (2022-12), SQLdeveloper
##### 📚 사용 언어 : Java, HTML 
##### 💡 구현 목표 : JSP로 통상적인 홈페이지의 게시판 기능 구현    
---

    

#### 📌 기본 기능
##### ▪️ DB에 등록된 회원 데이터 출력   
##### ▪️ 회원가입 기능 구현   
##### ▪️ 게시판 틀 작성     


---

#### 📌 게시판 기능 
##### ▪️ 답변 글은 들여쓰기로 표현   
##### ▪️ 게시글에 이미지 첨부 가능   
##### ▪️ 첨부한 사진 저장할 때 폴더에 글 번호별로 분리   
##### ▪️ 글 열람시 정보 출력   
##### ▪️ 수정 / 삭제 / 답글 기능   
##### ▪️ 글 페이지 10개 기준으로 나누기
